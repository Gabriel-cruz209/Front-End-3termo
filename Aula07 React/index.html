Introdução ao React & Vite

Aprenda os conceitos fundamentais de forma interativa

O que é o React?

É uma biblioteca JavaScript de código aberto focada na construção de interfaces de usuário (UI).

Diferencial: Em vez de ditar toda a estrutura, o React foca na visualização, permitindo criar partes isoladas (Componentes) e combiná-las.

Criado pelo Facebook, ele introduziu a mentalidade de Componentização no desenvolvimento web moderno.

Pilares Fundamentais

A. Componentização

Pense como peças de LEGO:

Botão
Input
Card
  • Reutilização: Use o mesmo componente em vários lugares.
  • Manutenção: Altere em um lugar, atualize em todos.

B. JSX (JavaScript XML)

Escreva HTML dentro do JavaScript de forma intuitiva:

const elemento = <h1>Olá Mundo!</h1>;

C. Virtual DOM

O React cria uma cópia leve do site na memória. Ele compara as mudanças e atualiza apenas o que é necessário no navegador real. Performance máxima!

Gerenciamento de Dados

Props (Properties): Dados que vêm de fora (pai para filho). São somente leitura.
State (Estado): A memória interna. Quando o estado muda, o componente se atualiza sozinho!

O que é o Vite? ⚡

Vite significa "Rápido" em francês. É a ferramenta que prepara seu ambiente de desenvolvimento.

  • Velocidade: Inicia o projeto instantaneamente usando Native ESM.
  • HMR: Atualiza apenas o que você mudou no código sem recarregar a página toda.
  • Produção: Deixa seu site leve e otimizado para os usuários.

Mão na Massa

1. Instalação

Instale o Node.js (LTS) e verifique no terminal:

node -v
npm -v

2. Estrutura de Pastas

  • node_modules/: Bibliotecas (Não mexer).
  • src/main.jsx: Onde tudo começa.
  • src/App.jsx: Seu componente principal.

3. Executando

npm run dev

Abra o link do localhost e edite o App.jsx para ver a mágica acontecer!
